Welcome to Super Simple Cloud!

To be able to use Super Simple cloud, you first have to adjust some settings.

You should set up auto-backup if you plan to store important files.

You should set up auto-recovery if you plan to store small files on flash drives/SD cards.

You should change the backup/recovery location if using a flash drive and/or SD card for storage. Default: .

You should change the Reed Solomon nysm to a smaller power of two if you plan to store big files.
